# Saberi3

Ово је најкраћи могући наставак примера `saberi`.

Сада функција не прима два, већ три `int` аргумента. Циљ је да се јасно види:

- да трећи аргумент стиже у `edx`
- да се повратна вредност и даље враћа преко `eax`
- да једноставна **leaf функција** и даље може да ради без стек оквира

## Датотеке

- `main.cpp` чита три броја и позива `saberi3`
- `saberi3.s` садржи асемблерску имплементацију функције

## Превођење

```sh
g++ main.cpp saberi3.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
2 4 7
13
```

## На шта треба обратити пажњу

- `a`, `b`, `c` стижу у `edi`, `esi`, `edx`
- нигде се не користи стек, јер функција нема локалне променљиве и не позива друге функције
- резултат се на крају смешта у `eax`
