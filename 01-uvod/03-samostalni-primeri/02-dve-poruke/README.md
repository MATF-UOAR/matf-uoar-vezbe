# Dve poruke

Ово је кратка вежба заснована на примеру `Hello world`.

Циљ је да се утврди:

- смештање константних ниски у `.rodata`
- рачунање адресе ниске инструкцијом `lea`
- позив `puts` више пута из исте функције
- поравнање стека пре `call`

## Датотеке

- `dve-poruke.s` садржи асемблерску имплементацију функције `main`

## Превођење

```sh
g++ dve-poruke.s
```

## Покретање

```sh
./a.out
```

Очекивани излаз је:

```text
Prva poruka
Druga poruka
```

## На шта треба обратити пажњу

- обе ниске су у `.rodata`, јер су константне
- `lea rdi, [rip+msg1]` и `lea rdi, [rip+msg2]` рачунају адресе ниски
- `puts` сам додаје нови ред, па ниске не садрже `\n`
- `push rbp` / `mov rbp, rsp` су довољни да стек буде поравнат пре оба позива функције `puts`
