# Самостални мини-примери

Ова целина садржи три кратка примера за самосталан рад после `Сабирања` и `Hello world`.

Идеја није да уводи потпуно нове концепте, већ да утврди:

- конвенцију позива
- рад са `eax`, `edi`, `esi`, `edx`
- основне аритметичке инструкције `add` и `sub`
- разлику између вредности и адресе
- позив функције `puts`
- секцију `.rodata` и поравнање стека пре `call`
