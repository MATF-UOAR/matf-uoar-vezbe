# Apsolutna vrednost

Ово је најкраћи могући задатак после примера `Максимум два броја`.

Функција `apsolutna_vrednost` прима један `int` и враћа његову апсолутну вредност. Циљ је да се утврди:

- поређење са нулом
- једна `if` грана и један условни скок
- инструкција `neg`

Референтна C++ верзија функције могла би да изгледа овако:

```cpp
int apsolutna_vrednost(int x) {
    if (x < 0) {
        return -x;
    }
    return x;
}
```

## Датотеке

- `main.cpp` чита број и исписује резултат
- `apsolutna_vrednost.s` садржи асемблерску имплементацију функције

## Превођење

```sh
g++ main.cpp apsolutna_vrednost.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
-7
|x| = 7
```

## На шта треба обратити пажњу

- аргумент `x` стиже у `edi`
- довољно је да упоредимо `x` са нулом и по потреби применимо `neg`
- као и у C++ коду са типом `int`, случај `INT_MIN` овде занемарујемо
