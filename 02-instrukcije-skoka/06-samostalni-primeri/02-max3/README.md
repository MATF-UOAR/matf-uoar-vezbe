# Max3

Овај пример је природан наставак примера `max`, али сада функција прима три броја.

Функција `max3` враћа највећи од три `int` аргумента. Циљ је да се утврди:

- да трећи аргумент стиже у `edx`
- како се тренутни максимум може чувати у `eax`
- како више узастопних `if` провера изгледа у асемблеру

Референтна C++ верзија функције могла би да изгледа овако:

```cpp
int max3(int x, int y, int z) {
    int rezultat = x;
    if (y > rezultat) {
        rezultat = y;
    }
    if (z > rezultat) {
        rezultat = z;
    }
    return rezultat;
}
```

## Датотеке

- `main.cpp` чита три броја и позива `max3`
- `max3.s` садржи асемблерску имплементацију функције

## Превођење

```sh
g++ main.cpp max3.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
4 9 2
max3(4, 9, 2) = 9
```

## На шта треба обратити пажњу

- `x`, `y`, `z` стижу у `edi`, `esi`, `edx`
- у `eax` можемо прво ставити `x`, па га касније по потреби ажурирати
- ово је добар пример за читање кода који има више кратких грана, али без `else` дела
