# Prestupna godina

Овај пример комбинује дељење, остатак и више узастопних грана.

Функција `je_prestupna` враћа `1` ако је година преступна, а `0` иначе. Циљ је да се утврди:

- поновна употреба шаблона `mov eax, ...`, `cdq`, `idiv`
- како се више услова проверава један за другим
- да се остатак из `edx` може користити више пута у истој функцији

Референтна C++ верзија функције могла би да изгледа овако:

```cpp
int je_prestupna(int godina) {
    if (godina % 400 == 0) {
        return 1;
    }
    if (godina % 100 == 0) {
        return 0;
    }
    if (godina % 4 == 0) {
        return 1;
    }
    return 0;
}
```

## Датотеке

- `main.cpp` чита годину и исписује резултат
- `prestupna_godina.s` садржи асемблерску имплементацију функције `je_prestupna`

## Превођење

```sh
g++ main.cpp prestupna_godina.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
2024
Prestupna godina
```

## На шта треба обратити пажњу

- пре сваког `idiv` поново припремамо `edx:eax`
- редослед провера је важан: `400`, па `100`, па `4`
- ово је поједностављено правило за грегоријански календар и сасвим је довољно за вежбу
