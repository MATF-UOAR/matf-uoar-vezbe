# Sezona

Ово је кратак `switch` задатак који уједно користи и `puts`.

Функција `sezona` прима број месеца и исписује одговарајућу сезону. Циљ је да се утврди:

- да више `case` вредности може да води у исти блок кода
- да `default` грана може да исписује посебну поруку
- рад са `.rodata`, `lea` и `puts` у функцији која грана извршавање

Референтна C++ верзија функције могла би да изгледа овако:

```cpp
void sezona(int mesec) {
    switch (mesec) {
    case 12:
    case 1:
    case 2:
        puts("Zima");
        break;
    case 3:
    case 4:
    case 5:
        puts("Prolece");
        break;
    case 6:
    case 7:
    case 8:
        puts("Leto");
        break;
    case 9:
    case 10:
    case 11:
        puts("Jesen");
        break;
    default:
        puts("Neispravan mesec");
    }
}
```

## Датотеке

- `main.cpp` чита број месеца и позива функцију `sezona`
- `sezona.s` садржи асемблерску имплементацију функције

## Превођење

```sh
g++ main.cpp sezona.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
4
Prolece
```

## На шта треба обратити пажњу

- ово је исти образац као у примеру са бројем дана у месецу, само сада `case` гране воде на различите позиве `puts`
- адресе ниски учитавамо са `lea rdi, [rip+...]`
- после једног исписа треба прескочити остале гране и отићи на крај функције
