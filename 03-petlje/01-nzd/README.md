# НЗД

Ово је први пример у ком видимо како се `while` петља ручно гради коришћењем инструкција скока.

## Шта програм ради

Програм учитава два `unsigned` броја и исписује њихов највећи заједнички делилац.

То може да се уради на више начина, али пошто је ове недеље фокус на петљама, користимо Еуклидов алгоритам. Референтна C++ функција могла би да изгледа овако:

```cpp
unsigned NZD(unsigned a, unsigned b) {
    while (b != 0) {
        unsigned temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

Иако `x86-64` има и специјалну `loop` фамилију инструкција, у пракси се петље много чешће пишу уз помоћ поређења, условних скокова и једног скока назад на почетак. У овом примеру петља се састоји из:

1. провере услова `(b != 0)`
2. тела петље
3. скока назад на почетак

## Шта треба посматрати у асемблеру

Проверу услова можемо да запишемо помоћу `cmp` и `je`:

```asm
while_petlja:
    cmp esi, 0
    je kraj_petlje
```

Када је `b == 0`, скачемо на излаз из петље.

Тело петље користи инструкцију `div`. Пошто радимо са `unsigned` вредностима, пре дељења морамо да поставимо вишу половину дељеника на нулу:

```asm
    mov eax, edi
    xor edx, edx
    div esi       # eax = a / b, edx = a % b

    mov edi, esi  # a = b
    mov esi, edx  # b = a % b
```

На крају тела петље безусловним скоком враћамо извршавање на почетак:

```asm
    jmp while_petlja
```

## Напомена о `enter` и `leave`

У коду ћете приметити да нисмо писали уобичајени пролог и епилог функције у дужем облику. Уместо:

```asm
push rbp
mov rbp, rsp
sub rsp, N
```

користимо:

```asm
enter N, 0
```

А уместо:

```asm
mov rsp, rbp
pop rbp
```

користимо:

```asm
leave
```

`enter` и `leave` су овде корисни као компактни запис исте идеје. То не значи да су нужно и најбољи избор у сваком реалном програму, у пракси ћете често виђати и експлицитан пролог и епилог.

## Датотеке

- `main.cpp` чита два броја и позива `NZD`
- `nzd.s` садржи асемблерску имплементацију функције `NZD`

## Превођење

```sh
g++ main.cpp nzd.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
12 18
NZD(12, 18) = 6
```

## На шта треба обратити пажњу

- `div` дели дељеник из `edx:eax`, па за `unsigned` дељење овде ручно `edx` попуњавамо нулама
- `while` петља у асемблеру се природно чита као „провера услова, тело, скок назад“
- `enter` и `leave` у овом примеру само скраћују запис стандардног пролога и епилога
