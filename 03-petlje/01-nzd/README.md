# NZD
Ово је први пример где видимо како би се имплементирала `while` петља коришћењем инструкција скока.

## Шта програм ради
Програм учитава два `unsigned` броја, и исписује њихов највећи заједнички делилац.
Ово може да се уради на више начина, али пошто је ове недеље фокус на петљама , користићемо
Еуклидов алгоритам. Референтна C++ функција би могла да изгледа овако:
```
int nzd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```
Како не постоје уграђене инструкције за петље у асемблеру, мораћемо да их изградимо из већ познатих инструкција.
Ако ближе погледамо структуру петље можемо да видимо да се она састоји из:

1. Провере услова `(b != 0)`
2. Самог тела функције
3. Скока назад на почетак кад дођемо до краја

Проверу услова (и сам услов) можемо да напишемо уз помоћ `cmp` и `je` инструкција које смо видели претходне недеље.
(Напомена: због једноставности се користи провера једнакости `(b == 0)`, да не бисмо бринули о беспотребним скоковима.
Ипак, ако читалац жели, ништа га не спречава да напише другачије програм. Могла би да буде добра вежба.)
```
while_petlja:
    cmp esi, 0
    je kraj_petlje
```
Види се да ако је услов испуњен излазимо из петље.
Тело програма се имплементира тривијално са инструкцијом  `div`. Но, битно је напоменути да се програм прима `unsigned` бројеве,
тако да је нопходно анулирати `edx` регистар пре саме операције дељења.
```
   mov eax, edi
    xor edx, edx
    div esi       # eax = a / b edx = a % b

    mov edi, esi  # a = b
    mov esi, edx  # b = a % b
```
И на самом крају тела налази се један безусловни скок који нас враћа назад на почетак петље.
```
jmp while_petlja
```
## Напомена
Кад видите код приметићете да нисмо користили наш стандардан шаблон за намештање стек оквира.
Пошто се тај код често понавља, уведене су инструкције које раде исту ствар, ради ефикасности.
Тако
```
  push rbp
  mov rbp, rsp
  sub rsp, N     
```
може да се замени са
```
  enter N, 0
```
На исти начин
```
    mov rsp, rbp         
    pop rbp
```
може да се замени са
```
  leave
```
# Датотеке
- `main.cpp` чита два броја и позива нзд
- `nzd.s` садржи асемблерску имплементацију функције
# Превођење
  ```
g++ main.cpp nzd.s
  ```
# Покретање
```
./a.out
```
Пример интеракције:
```
12 18
NZD(12, 18) = 6
```
