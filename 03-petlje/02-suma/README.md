# Сума
На овом примеру ћемо видети као се имплементира `for` петља, и како се користи инструкција за множење `mul`
## Шта програм ради
Програм учитава `unsigned` број `n` и исписује збир првих `n` природних бројева.
Референта С++ функција може да изгледа овако:
```
unsigned suma(unsigned n) {
  unsigned trenutna_suma=0;
  for(unsigned i=1;i<=n;i++)
  {
    trenutna_suma+=i;
  }
  return trenutna_suma;
   
}
```
Структура наше `for` петље је скоро идентична као структура `while` петље, једина разлика је што пре почетка морамо
да иницијализујемо индекс (`i=1`) и  на крају тела морамо да инкрементирамо тај индекс (`i++`).
Прво одредимо регистар да чува тренутну суму и ининцијализујемо индекс:
```
    xor eax, eax   # eax - trenutna suma
    mov ecx, 1     # ecx - i
```
Након тога у петљи прво проверавамо услов излаза, и искачемо ако је испуњен:
```
  for_petlja:
      cmp edi, ecx
      jb  for_petlja_kraj
```
И унутар тела повећавамо суму, инкрементирамо индекс са наредбом `inc` и безусловним скоком се враћамо на почетак:
```
  add eax, ecx

    add ecx, 1
    jmp for_petlja
```
## Имплементација без петље
Неки читаоци су вероватно до сад помислили да сума првих `n` природних бројева може да се израчуна много лакше, без петље.
И били би у праву. Сума првих `n` природних бројева је једнака `n*(n+1)/2`. Да би смо овај израз имплементирали у асемблеру потребна
нам је инструкција `mul`. Инструкција `mul` слично као инструкција `div` има један операнд, и њега множи са вредности у
регистру `rax`. Резултујући производ је смештен у `rdx:rax` (тјст. `edx:eax` у случају четворобајтних бројева). Битно је
напоменути да ће након множења вредност регистра `rdx` да се мења, тако да ако се ту налази нешто битно, требало би га сачувати
пре инструкције множења.
```
    mov eax, edi
    add edi, 1
    mul edi     # eax = n * (n+1)

    mov esi, 2
    xor edx, edx
    div esi     # eax = n * (n+1) / 2

```
Напомена: Слично као за инструкције дељења, постоји инструкција `imul` која служи за множење вредности у потпуном комплементу, а
`mul` само служи за множење неозаначених бројева.


## Датотеке
- `main.cpp` чита један број и позива суму
- `suma.s` садржи асемблерску имплементацију функције са петљом
- `suma_proizvod.s` садржи асемблерску имплементацију функције без петље
## Превођење
 За верзију са петљом:
  ```
g++ main.cpp suma.s  
  ```
За верзију без петље:
```
g++ main.cpp suma_proizvod.s  

```
## Покретање
```
./a.out
```

