# Сума парних бројева

Овај пример сабира све парне бројеве од `1` до `n` и служи као увод у инструкције `test` и `loop`.

## Шта програм ради

Програм учитава `unsigned` број `n` и исписује збир свих парних бројева из интервала `1..n`.

Референтна C++ функција може да изгледа овако:

```cpp
unsigned suma_parnih(unsigned n) {
    unsigned s = 0;
    for (unsigned i = 1; i <= n; i++) {
        if (i % 2 == 0) {
            s += i;
        }
    }
    return s;
}
```

У овом фолдеру постоје две асемблерске верзије исте функције:

- `suma_parnih_for.s` је имплементација слична имплементацији `for`-петљом из [другог задатка](../02-suma/README.md)
- `suma_parnih_loop.s` је имплементација исте функције уз помоћ инструкције `loop`

## Верзија 1: класична петља

У првој верзији идемо од `1` до `n`, као у C++ коду:

```asm
    xor eax, eax
    mov ecx, 1

for_petlja:
    cmp edi, ecx
    jb  for_petlja_kraj
```

Парност не морамо да проверавамо дељењем, јер зависи само од најнижег бита: код парног броја он је `0`, а код непарног `1`. Зато овде користимо инструкцију `test`, која нам омогућава да то проверимо битовском операцијом:

```asm
    test ecx, 1
    jnz nastavak
    add eax, ecx
```

## Верзија 2: Инструкција `loop`

Друга верзија прати идеју:

```cpp
unsigned suma_parnih(unsigned n) {
    unsigned s = 0;
    unsigned i = n;

    if (i == 0) {
        return 0;
    }

    do {
        if ((i & 1) == 0) {
            s += i;
        }
        i--;
    } while (i != 0);

    return s;
}
```

У асемблеру то изгледа овако:

```asm
    xor eax, eax
    mov ecx, edi
    test ecx, ecx
    je kraj

do_while_petlja:
    test ecx, 1
    jnz nastavak
    add eax, ecx

nastavak:
    loop do_while_petlja
```

Инструкција `loop` је кратак запис за:

1. декрементирај `ecx`
2. ако `ecx != 0`, скочи назад

Зато је природна за овакву силазну петљу. Битан детаљ је почетна провера `n == 0`: без ње би `loop` за нулу направио погрешно понашање, јер би после декремента бројач прешао на велику неозначену вредност.

## Датотеке

- `main.cpp` чита број и позива `suma_parnih`
- `suma_parnih_for.s` садржи класичну верзију са експлицитном провером услова петље
- `suma_parnih_loop.s` садржи силазну do-while-стил варијанту са `loop`

## Превођење

За класичну верзију:

```sh
g++ main.cpp suma_parnih_for.s
```

За `loop` верзију:

```sh
g++ main.cpp suma_parnih_loop.s
```

## Покретање

```sh
./a.out
```

Пример интеракције:

```text
10
30
```

## На шта треба обратити пажњу

- `test reg, 1` је згодан начин да проверимо да ли је број паран
- `loop` имплицитно користи бројач у `ecx`, па је погодан само када нам тај регистар слободно стоји на располагању
- do-while-стил петља природно креће од тела, али код `unsigned` бројача морамо посебно пазити на случај `n = 0`
