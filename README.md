# УОАР – материјали са вежби

Овај репозиторијум садржи материјале за практични део курса **„Увод у организацију и архитектуру рачунара“**, који се слуша на првој години И смера на Математичком факултету.

Кроз материјале се обрађују основни концепти програмирања у асемблеру за архитектуре процесора `x86-64` и `ARM64`.

Материјали укључују примере, задатке и пратећи код који се користи на вежбама.

Репозиторијум је у активној изради, па су све сугестије, исправке и коментари добродошли.
